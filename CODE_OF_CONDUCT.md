# Code of Conduct

This project expects respectful, constructive collaboration.

## Expected behavior

- Use inclusive and professional language.
- Focus criticism on ideas and code, not individuals.
- Assume good intent and communicate clearly.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Discriminatory or abusive language.
- Sharing private information without consent.

## Reporting

Report conduct issues privately to `ismail.elkorchi@gmail.com`.
For security issues, use channels in `SECURITY.md`.
